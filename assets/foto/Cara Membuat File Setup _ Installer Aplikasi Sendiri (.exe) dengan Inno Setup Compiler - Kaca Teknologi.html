    Cara Membuat File Setup / Installer Aplikasi Sendiri dengan Inno Setup Compiler


    Pernahkah anda bertanya-tanya bagaimana caranya membuat file setup / installer / installan aplikasi sendiri dalam berbentuk .exe ? kemungkinan anda akan menemukan hal ini ketika akan menginstall sebuah aplikasi dan anda diharuskan untuk mengklik next-next dan next agar aplikasi tersebut terinstall. Namun, ketika anda membuat sebuah aplikasi menggunakan Visual Studio (misalnya) semua isi file tersebut akan berantakan dan pada waktu inilah anda perlu sebuah aplikasi untuk merapihkan file-file yang berantakan tersebut menjadi satu package installer sehingga pengguna yang akan menginstall aplikasi anda pun tidak akan kebingungan.

     Adapun nama aplikasinya adalah Inno Setup Compiler.
    Inno Setup adalah sebuah perangkat lunak instalasi bebas berbasis skrip yang ditulis menggunakan Embarcadero Delphi oleh Jordan Russel. Versi pertama perangkat lunak ini dirilis tahun 1997. Aplikasi ini memudahkan para programmer untuk menyatukan beberapa file penting agar dapat terinstall dengan mudah.

    Bagaimana Cara Membuatnya ? 

    Cara membuat file setup / installer aplikasi sendiri (.exe) dengan Inno Setup Compiler adalah sebagai berikut...
    1. Pertama, unduh aplikasinya di sini → Download Inno Setup Compiler.
    2. Kedua, install dan jalankan aplikasi tersebut.
    3. Ketiga, setelah aplikasi dijalankan pilih File → Create a new empty script file atau bisa dengan shortcut CTRL + N kemudian klik next.
    4. Keempat, pada bagian ini anda bisa isi spesifik dari aplikasi anda. Seperti nama, versi, penerbit, dan website aplikasi.
    5. Kelima, lakukan seperti gambar di bawah. Application destination base folder adalah folder tujuan dari aplikasi, anda bisa mengubahnya atau membiarkannya saja. Jika anda ingin agar orang lain bisa mengubah folder hasil instalan aplikasi anda, anda bisa centang "Allow user to change the application folder" atau jika aplikasi anda tidak membutuhkan folder anda bisa centang "The application doesn't need a folder".
    6. Keenam, pilih file yang akan dijadikan installer dengan klik browse jika aplikasi anda berformat exe atau centang "The application doesn't have a main executable file" jika yang akan dijadikan installer adalah sebuah folder atau file yang bukan berformat .exe.
    7. Ketujuh, centang semua kotak kecil untuk memudahkan pengguna.
    8. Kedelapan, pada bagian ini seperti lisensi, informasi sebelum instal dan sesudah instal diatur anda bisa memasukkannya dengan cara membuat terlebih dahulu file yang berbentuk .txt dan tentunya di dalamnya sudah disisipi kata-kata. Kemudian masukkan file-file tersebut dengan klik browse.
    9. Kesembilan, pilih bahasa apa saja yang tersedia untuk aplikasi anda.
    10. Kesepuluh, anda bisa mengatur output folder, nama setup, ikon setup, dan kata sandi setup.
    11. Kesebelas, centang "Yes, use #define compiler directives".
    12. Keduabelas, pembuatan setup selesai anda bisa klik finish.
    13. Ketigabelas, klik ikon seperti gear pada bagian atas kemudian tunggu sampai selesai. Untuk lebih jelasnya anda bisa lihat gambar di bawah.
    14.  Selesai, anda bisa cek kembali output folder anda dan coba jalankan setupnya.
